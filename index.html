Connect 4

The game starts with a welcome screen offering two options: "Player vs Player" and "Player vs AI". If the player selects "Player vs Player", the game proceeds to ask for the names of both players.If the player selects "Player vs AI", the game proceeds to ask for the player's name. After names are entered, the game asks each player which color they prefer: red or yellow. Once colors are chosen, the game begins with an empty Connect 4 grid/board. The game prompts the player whose turn it is to make a move. After each move, the game checks if there is a winner or if the board is full (draw). If there's a winner, it announces the winner. If it's a draw, it asks if the players want to play again or quit. If players choose to play again, the game resets with a new empty board and the same players. If players choose to quit, the game closes.

File Manager

This project is like a digital organizer that helps you manage your files and folders. It uses a few tools to do this, like opening and moving files (using 'os' and 'shutil') and organizing data (with 'sqlite3'). When you start the program, it sets up a special storage place called a SQLite database to keep track of your folders. Then, it creates a table in this database to store the names of the folders you create. Once everything is set up, you can start using the program. It gives you options like creating a new folder, copying files from one place to another, and sorting your folders based on their names or sizes. If you want to see what files are inside a folder, it can do that too. To use it, you just follow the instructions it gives you on the screen. It's like having a helpful assistant for managing your computer files.
